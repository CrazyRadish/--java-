（1）选择Help-----》Install from site-----》点击Add，弹出对话框中，Name输入egit,location输入http://download.eclipse.org/egit/updates，点击OK

（2）选择如下，点击Next，耐心等待，可能时间较长，之后Myeclipse会自动重启


（3）选择Windows---》Preferences---->SSH2---->KeyManagement---->Generate RSA Key---->Apply---->Save

（4）回到Github网站，选择如下，Add SSH key，将上图生成的ssh-rsa复制进去

（5）选择自己的项目，右键，选择如下

（6）创建仓库，点击Finish

（7）右键项目，选择Team，选择Commit

（8）添加描述，选中所有的文件，点击Commit and Push

（9）弹出如下对话框，（不要关掉，看下一步）

（10）回到Github网站，创建一个仓库，将下面框中uri复制到上面的对话框，这里MyEclipse如果选择HTTPS的地址它要填写端口，我试了在Eclipse中并不需要，所以这里选择下下张图里SSH的地址（我用的是MyEclipse专业版2014，不知别的版本有没有问题）

（10）等待Push.....，它会让你填写一些问题和答案，随意填写


（11）这是在GitHub上的仓库里就出现了项目内容
